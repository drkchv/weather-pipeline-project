# Weather Data Engineering Project

## Описание проекта
ETL-пайплайн для сбора данных о погоде через API, их обработки и визуализации. 
Проект включает полный цикл Data Engineering: от получения данных до построения дашбордов.

## Архитектура
- **Источник данных**: WeatherAPI.com
- **Оркестрация**: Apache Airflow
- **Хранилище**: PostgreSQL
- **Трансформация**: Python (Pandas)
- **Визуализация**: Redash
- **Инфраструктура**: Docker

## Функциональность
- Сбор данных о погоде каждый час
- Поддержка 6 городов (Санкт-Петербург, Москва, Минск, Лондон, Токио, Нью-Йорк)
- Текущая погода и прогноз на 3 дня
- Автоматическая загрузка в PostgreSQL
- Интерактивный дашборд в Redash
